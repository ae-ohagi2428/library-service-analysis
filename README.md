# 「財政力があれば図書館は充実する」は本当か――544自治体の探索的データ分析

公共図書館544自治体のオープンデータから、サービス水準を左右する要因を探った探索的データ分析。

---

## Ⅰ　分析の概要
人口規模5万人以上自治体の公共図書館データ544件を対象に、図書館サービスに影響を与える要因について、相関・回帰・残差などの観点から比較・分析を行った。なお、図書館サービス水準を示す指標は「市民1人あたりの貸出数」とした。

---

## Ⅱ　分析結果
本分析によって明らかとなった内容は以下３点のとおり。
①市区町村の「財政力指数」だけでは「市民1人あたりの貸出数」を充分に説明することはできない。
②「財政力指数」よりも「市民1人あたりの資料費」のほうが、「市民1人あたりの貸出数」の説明力が高い（相関が強い）。
③「奉仕人口」と「市民1人あたりの貸出数」に相関関係はほぼ見られない。ただし人口規模が大きくなるほど変動係数は小さくなり、小〜中規模では突出して貸出数の多い自治体が存在する一方、大規模ではそうした突出が見られない。

---

## Ⅲ　分析の背景
筆者は9年の行政経験の中で、図書館サービス向上の業務に携わってきたことから、
図書館サービスの水準を左右する要因をデータから検証したいと考えた。

   https://www.soumu.go.jp/iken/zaisei/R01_chiho.html
   （令和8年6月20日に利用）
   対象：1の市区町村　544件

## Ⅵ　分析手順
「図書館サービス水準に影響を与える要因は"自治体の財政力"である」という仮説（仮説①）を設定。必要なデータを取得・加工し、「財政力指数」と「市民1人あたりの貸出数」の相関分析を行った。結果からは仮説①を充分に実証できなかったため、外れ値の検証を行い、「図書館サービス水準に影響を与える要因は"市民1人あたりの資料費"である」という仮説（仮説②）と「奉仕人口の規模によって、市民1人あたりの貸出数に何らかの傾向がある」という仮説（仮説③）を再度設定。仮説②では「市民1人あたりの資料費」と「市民1人あたりの貸出数」の相関を分析した。仮説③では「市民1人あたりの貸出数」と「奉仕人口」の関係を散布図で示した上で、自治体の人口規模を「(5万人以上)30万人未満」「30万人以上50万人未満」「50万人以上」で区切って比較を行った。
なお、詳細についてはZenn記事Ⅸを参照。

## Ⅷ　再現手法
### 動作環境
- Python 3.13.5
- SQLite (DB Browser for SQLiteを使用)
- 主要ライブラリ：pandas, numpy, matplotlib, scipy, japanize-matplotlib

### 手順
1. データの取得
Ⅳに記載した出典より、以下をダウンロードし、data/ に配置。
・「日本の図書館 統計と名簿 2019年版」（日本図書館協会）
・「全市町村の主要財政指標（令和元年度）」（総務省）

2. データベースの作成
DB Browser for SQLite で新規DBを作成。
上記2ファイルを文字コードUTF-8で、それぞれ library_stats_2019 / fiscal_index_2019 テーブルとして取込。
profiling.sql を実行し、ビュー v_library_over50k を作成。

3. 分析の実行
analysis.ipynb を上から順に実行。
